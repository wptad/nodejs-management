## node-icu-charset-detector

```
  brew install icu4c
  ln -s /usr/local/Cellar/icu4c/<VERSION>/bin/icu-config /usr/local/bin/icu-config
  ln -s /usr/local/Cellar/icu4c/<VERSION>/include/unicode /usr/local/include

```
